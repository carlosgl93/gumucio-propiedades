# Proyecto: Sitio Web para Agente Inmobiliaria

## Cliente

Una agente inmobiliaria ubicada en Rancagua, Chile. Todos los textos visibles al usuario deben estar en español.

## Objetivo

Crear un sitio web para promocionar propiedades en venta o arriendo, con una interfaz moderna y minimalista.

### Página Principal (Home)

El diseño visual ya está definido (ver imagen adjunta). La página de inicio debe incluir:

- Logo al centro superior: “Gumucio Propiedades”
- Imagen de fondo de una casa moderna
- Botón: “VENDE O ARRIENDA TU CASA”
- Botón: “COMPRA O ARRIENDA”
- Llamado principal: “¿QUIERES EMPEZAR UN CAPÍTULO NUEVO?”
- Tarjetas con propiedades destacadas (estado, descripción, detalles como habitaciones/baños/garage)

### Requerimientos Técnicos

- Stack: Vite + React + TypeScript
- Backend: Firebase (Firestore para base de datos, Firebase Auth, Firebase Hosting)
- El sitio será eventualmente escalado para incluir más funcionalidades como búsqueda, filtros, y contacto.

## Estilo UI

- Minimalista y profesional
- Tipografía serif para títulos
- Colores neutros (blanco, negro, verde claro)
- Enfoque en usabilidad, claridad de la información y visualización de propiedades

## Consideraciones

- Todo el contenido debe estar en español
- Enfocado en usuarios de Rancagua, Chile
- Las propiedades pueden estar “en venta” o “en arriendo”
- Debe ser responsivo

## Copilot

Cuando generes componentes o funciones:

- Usa nombres de variables, componentes y clases en inglés
- Usa textos visibles al usuario en español
- Sigue el diseño de la imagen proporcionada como referencia
